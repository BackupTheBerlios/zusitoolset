00 04 - ACK_NEEDED_DATA

Zusi » Server und Server » Externe Anwendung
Entscheidung, ob die benötigten Daten akzeptiert werden. Der Befehl wird nach jedem NEEDED_DATA-Befehl gesendet.

Länge Inhalt Bezeichnung Beschreibung
4 bytes xx xx xx xx PACKET_LENGTH Länge des nachfolgenden Datenpakets in Byte (Cardinal, Intel-Order)
2 bytes 00 04 ACK_NEEDED_DATA Befehl ACK_NEEDED_DATA
1 byte xx ACK Der Befehl wurde akzeptiert, wenn das Byte auf 0x00 steht. Wird der Befehl nicht akzeptiert, wird stattdessen ein anderes Byte gesendet.
Das Byte kann einen Fehlercode enthalten:
0x01: Unbekannter Befehlsvorrat
0x02: Client nicht angemeldet (HELLO)
