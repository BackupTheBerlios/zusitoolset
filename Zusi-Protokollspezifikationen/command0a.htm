00 0A - DATA

Zusi » Server und Server » Externe Anwendung
Der DATA-Befehl 00 0A überträgt Nutzdaten aus dem Befehlsvorrat "Führerstandsinstrumente
Andere Data-Befehle werden mit den nachfolgenden Bytes referenziert, wenn die IDs für 00 0A nicht mehr ausreichen.

Länge Inhalt Bezeichnung Beschreibung
4 bytes xx xx xx xx PACKET_LENGTH Länge des nachfolgenden Datenpakets in Byte (Cardinal, Intel-Order)
2 bytes 00 0A DATA Befehl DATA
variabel xx ... ID_DATA Ein ID-Byte, gefolgt von der festgelegten Länge des ID-Wertes. Wenn die Länge variabel ist, folgt auf das ID-Byte ein Längenbyte und darauf die Daten (z.B. bei Strings)
