Topologie

Der Server ist die Schnittstelle zwischen Zusi und externen Komponenten, wie Fahrpulten, Elektronischen Buchfahrplänen oder auch Drucktastenstellwerken. Alle externen Komponenten melden sich am Server an und vereinbaren, welche Daten sie benötigen. Zusi meldet sich ebenfalls am Server an. Der Server vereinbart nun mit Zusi, welche Daten insgesamt benötigt werden. Diese Daten werden von Zusi in loser Reihenfolge zu der Zeit, zu der sie bereitstehen, an den Server gesendet, dieser kümmert sich um die korrekte Aufsplittung der Daten und sendet sie dann an die entsprechenden Komponenten weiter.
